OpenShot Video Editor Introduction

Step 1 - Import Files

1.1. Files can be dragged and dropped on to the 'Project Files' section.

1.2. Files can be imported by clicking on the import button on the toolbar.

1.3. Files can be imported via the menu bar: Go to File - Import Files.

Step 2 - Add and Arrange Files on the Timeline

2.1. The files on 'Project Files' should be dragged onto track(s) in the timeline in order to view them. On the timeline, the video or images can be arranged in whatever viewing sequence you require. A combination of files can be added on to the tracks, for example: the first track can be used for images and the next track can be used to add music. The same file can be added on multiple times to the track

Changing the size of the files on the tracks (by dragging the edges of the file to make it shorter or longer) will determine the duration of play. Overlapping images will create a fading tranisition effect. The blue rounded rectangles between the clips indicate that a fading transition will take place.

Step 3 - Preview Project

3.1. Click on the 'Play' button under the 'Video Preview' section to preview your project.

In addition to playing the video, you can also pause, rewind, and fast-foward your video project (if required) by clicking on the corresponding buttons.

Step 4 - Export Project

4.1. Once the project is complete, you can export the project. The converted file can be played on most media players or websites. In order export the video, click on the export button on the toolbar.

When you export the video, you will have to select the preferred export options from a pop-up window.

4.2. Alternatively, you can export videos using the menu-bar. Go to File - Export Project - Export Video.
