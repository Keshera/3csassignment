OpenShot Video Editor Introduction

Video Tutorial

1. Main Window

Number Name Description
1 Main Toolbar The 'main toolbar' contains buttons to open, save, and export your video project.
2 Project Files All audio, video, and image files that have been imported into your project will be shown on the 'Project Files' section.
3 Preview Window The 'preview window' is the area that the video will playback on the screen.
4 Edit Toolbar The 'edit toolbar' contains buttons used for snapping, inserting markers, and jumping between markers.
5 Zoom Slider The 'zoom slider' will adjust the time-scale of your timeline.
6 Play-head / Ruler The ruler shows the time-scale, and the red line is the play-head. The play-head represents the current playback position.
7 Timeline The timeline visualizes your video project, and each clip and transition in your project.
8 Filter Filter the list of items shown (project files, transitions, and effects) by using these buttons and filter textbox. Enter a few letters of what you are looking for, and the results will be shown.
9 Playback Left to Right: Jump to Start, Rewind, Play/Pause, Fast Forward, and Jump to End

2. Built-in Tutorial

2.1. When you first launch OpenShot, you will be presented with a built-in tutorial. It will provide a basic overview of OpenShot Video Editor. Clicking Next will jump to the next topic. If required, You can view this tutorial again from the Helpâ†’Tutorial menu.

3. Import Files

3.1. Files can be dragged and dropped on to the 'Project Files' section.

3.2. Files can be imported by clicking on the import button on the toolbar.

3.3. Files can be imported via the menu bar: Go to File - Import Files.

4. Add and Arrange Files on the Timeline

4.1. The files on 'Project Files' should be dragged onto track(s) in the timeline in order to view them. On the timeline, the video or images can be arranged in whatever viewing sequence you require. A combination of files can be added on to the tracks, for example: the first track can be used for images and the next track can be used to add music. The same file can be added on multiple times to the track

4.2. Changing the size of the files on the tracks (by dragging the edges of the file to make it shorter or longer) will determine the duration of play. Overlapping images will create a fading tranisition effect. The blue rounded rectangles between the clips indicate that a fading transition will take place.

OpenShot uses tracks to layer videos and images. The top most track is the top layer (clips on this track will be visible) and the bottom track is the bottom layer. OpenShot will mix each layer together. You can have an unlimited number of tracks, but typically a video project will not need more than 10 (or so) tracks.

5. Preview Project

5.1. Click on the 'Play' button under the 'Video Preview' section to preview your project.

In addition to playing the video, you can also pause, rewind, and fast-foward your video project (if required) by clicking on the corresponding buttons.

6. Export Project

6.1. Once the project is complete, you can export the project. The converted file can be played on most media players or websites. In order export the video, click on the export button on the toolbar.

When you export the video, you will have to select the preferred export options from a pop-up window.

6.2. Alternatively, you can export videos using the menu-bar. Go to File - Export Project - Export Video.
